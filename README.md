# Diatonic_Chord_Key_Finder
入力されたコード進行から、そのコードが含まれる可能性のあるダイアトニックキー（調）を判定するPythonツールです。

## 概要
このスクリプトは、与えられた複数のコードを受け取り、

それらすべてを「ダイアトニックコード」として含むメジャーキーおよびマイナーキーをリストアップします。

作曲の際のキー特定や、耳コピの補助ツールとして利用できます。


## 特徴
- **シャープ（#）表記への統一**: 音楽理論上のフラット表記を避け、すべてシャープに統一して判定を行います（例：Bb → A#）。

- **メジャー/マイナー両対応**: 12のメジャーキーと12のナチュラルマイナーキーを網羅。
  
- **シンプルな判定ロジック**: 入力されたコードが各キーのコードセットの部分集合であるかを判定します。
  

## 使い方

1. `diatonic_finder.py` をダウンロードまたはクローンします。
   
3. 以下のようにコードをリスト形式で渡して実行します。

```python

from diatonic_finder import DiatonicKeyFinder

finder = DiatonicKeyFinder()

# 調べたいコードを入力

keys = finder.find_keys(["C", "Dm", "G"])

print(keys)

# 出力例: ['C Major', 'A Minor']
